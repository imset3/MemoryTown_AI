# Weekly Assignment

이 폴더는 5주차 weekly assignment를 final assignment와 구분해서 정리한 공간입니다.

## 요약

- Weekly assignment 안내: 파일 참조
- Final assignment 안내: AI를 활용한 스타트업 서비스 기획 및 구현
- 강의 질의응답 메모: final assignment는 시험 내용과 연관된 것으로 안내됨

## 확인할 내용

- Weekly assignment는 별도 구현물이 아니라, 강의에서 제공된 파일을 참고하는 성격입니다.
- 실제 구현 과제는 [`../final_assignment/`](../final_assignment/)에 독립 실행 가능한 형태로 분리했습니다.
- 교수님이 구현물과 기획서를 확인하실 때는 `final_assignment/`를 먼저 보면 됩니다.

## 관련 메모

자세한 정리는 [weekly_assignment_note.md](./weekly_assignment_note.md)에 적어두었습니다.
