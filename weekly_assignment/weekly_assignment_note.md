# 5주차 Weekly Assignment 정리

## 강의 공지 요약

### Weekly assignment

- 안내 내용: 파일 참조
- 별도 앱 구현 요구가 아니라, 제공된 파일과 과제 세부사항을 확인하는 성격으로 이해했습니다.

### Final assignment

- 과제명: AI 스타트업 프로젝트 과제
- 주제: AI를 활용한 스타트업 서비스 기획 및 구현
- 결과물: 기획서 또는 PPT 문서 + 구현물
- 구현 플랫폼: 웹, 앱, 스크립트, Streamlit 중 택 1
- 필수 조건: AI 기능 1개 이상 실제 작동
- 데이터: 자체 제작 가능

## 질의응답 메모

수업 시작 전 final assignment가 사전에 공지된 시험과 관련이 있는지 질문했고, 강사님 답변은 “예 시험 내용입니다.”였습니다.

따라서 이 저장소에서는 final assignment를 단순 예제 수준이 아니라, 시험/평가용 제출물로 볼 수 있게 별도 폴더에 독립 구성했습니다.

## 이 저장소에서의 분리 방식

- `weekly_assignment/`: weekly assignment 및 수업 공지 메모
- `final_assignment/`: 최종 제출용 Streamlit 구현물, 기획서, 발표자료, 학습자료, 테스트, 실행 스크립트

## 교수님 확인 경로

최종 구현물은 아래 폴더에서 바로 확인할 수 있습니다.

```text
final_assignment/
```

실행 방법:

```bash
cd final_assignment
pip install -r requirements.txt
streamlit run app.py
```
